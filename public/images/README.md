# Profile Images

Add the two profile images to this folder:

1. **john.jpg** - Left photo on the start screen
2. **elly.jpg** - Right photo on the start screen

## Requirements

- Images should be square or portrait orientation
- Recommended size: at least 400x500px
- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`

## Usage

Simply drop your image files into this folder with the exact filenames above. The app will automatically display them on the "You Matched!" start screen.

```
public/
└── images/
    ├── john.jpg    <-- add this
    ├── elly.jpg    <-- add this
    └── README.md
```
